QUELQUES POINTS DE BRODERIE

Découvrer quelques points parmi les plus courants

  • Boulogne
    X CLOSEe

    Point de boulogne

    Le point de Boulogne, ou point de couchure, se réalise avec plusieurs fils. Le ou les fils posés sur le tissu sont maintenus par un autre fil, généralement plus fin, en points réguliers

  • Palestrina

    Point de palestrina

    Le point de Palestrina est un point originaire d’Italie utilisé principalement pour entourer des motifs ou pour des bordures. C’est un point qui apporte beaucoup de relief au travail.

    X CLOSE
  • Plume

    Point de plume

    Ce point s'utilise pour broder les petites fleurs ou les herbes.

    X CLOSE
  • Feston

    Point de feston

    Le point de feston ressemble au point utilisé pour les boutonnières, mais les points sont plus espacés et doivent absolument être très réguliers.

    X CLOSE
  • Chaînette

    Point de chaînette

    Il est tout aussi utile pour remplir des formes que pour créer des contours, car sa structure en chaine signifie qu'il est assez flexible pour suivre des courbes et des spirales.

    X CLOSE
  • Bouclette

    Point de bouclette

    La bouclette aussi appelé point de marguerite ou point de chaînette isolé est idéal pour créer de petites feuilles ou les pétales d'une fleur.

    X CLOSE
  • Grébiche

    Point de grébiche

    Le point de grébiche est un point de base en broderie servant principalement à réaliser des bordures décoratives. L'avantage de ce point est que sa partie horizontale empêche le tissu de s'effilocher.

    X CLOSE
  • Nœud

    Point de nœud

    Le point de nœud est le plus simple des points noués. Il produit de jolis effets de plumetis (broderie en relief) ou de grains.

    X CLOSE
  • Chausson

    Point de chausson

    Le point de chausson est un point de croix décoratif. Il permet de créer une ligne en zig-zag pour embellir une bordure ou une couture.

    X CLOSE
  • Passé plat

    Point de
    passé plat

    Le passé plat est un point de remplissage. Il consiste à broder des points très rapprochés de façon à recouvrir totalement le tissu.

    X CLOSE
